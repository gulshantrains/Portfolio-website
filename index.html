🎬 MovieMind: Intelligent Movie Rating & Recommendation System

Developed an end-to-end machine learning system that predicts IMDb ratings based on director, actors, genres, and duration, and recommends movies using genre similarity. Utilized TMDb API and fuzzy matching for movie title inconsistencies.

  • 🔍 Features: Rating prediction, genre-based recommendations, cold-start handling
  • 🛠️ Technologies: Python, Pandas, Scikit-Learn, TMDb API, FuzzyWuzzy, XGBoost
  • 📊 Dataset: The Movies Dataset created by combining various online sources
